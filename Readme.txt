<h1>Neobis_Shortener_Project </h1>

+ Описание проекта

В проекте реализована возможность создания короткой ссылки.
- **URL Shortener**
    - Было создано django-приложение и настроино базу данных-postgres.
    - Cоздаyj модель для хранения ссылок
    - Создано URL ссылку на страницу с формой для ввода ссылки с кнопкой запроса ,Кнопка, отправляет ссылку на сервер для обработки
    - Написано view django для обработки запросов на сокращение ссылок. Во view получино ссылку из запроса,было сгенерировано уникальный идентификатор для ссылки и сохранить его в базе данных - в serializer.
    - Затем нужно сформирование короткой ссылку, содержащую эту ссылку
    - Добавлено функция для преобразования короткой ссылки обратно в исходную длинную ссылку

<h1>Установка</h1>

+ Склонируй репозиторий используя команду

git@github.com:https://github.com/Argen2002/Neobis_Shortener_Project
+ Создай виртуальное окружение используя команду

python3 -m venv <name of your environment>
+ Активируй виртуальное окружение

source <name of your environment>/bin/activate
+ Установи зависимости

pip install -r requirements.txt
+ Создай .env файл


+Убедитесь что у вас установлен Postgresql/в проекте используется postgresql/если его нет попробуйте:
-скачать postgresql/yстановить - создать в PgAdmin 4 новую базу данных нажмите сохранить пароль и название базы даных и поменяйте название в настройках shortenrproject

+ Зайди в директорию config

cd config
+ Сделай миграции

  ./manage.py migrate
+ Запусти свой проект

./manage.py runserver
